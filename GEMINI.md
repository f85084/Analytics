# Stock Dashboard Project Instructions

## Workflows
- **Update Data**: When the user says "執行更新數據", run the integrated update tool:
  \`node /data/anna/aider/tools/update-stock-data.js\`

## Technical Standards
- The project uses a card-based UI with a stable CSS Grid layout.
- Recommendations are based on Bollinger Band position and SMA slope.
- Explanations should be in plain language (Traditional Chinese).
