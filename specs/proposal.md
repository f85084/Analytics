# Proposal: Stock Dashboard Evolution

## 1. Problem Statement
The current dashboard relies on a single set of technical indicators (Bollinger Bands & SMA) which may be insufficient for comprehensive market analysis. Users find raw data difficult to interpret without visual aids. Additionally, there is no way to track "what-if" scenarios (mock trading) across different devices (PC and mobile).

## 2. Proposed Solution
Upgrade the dashboard into an "AI Investment Assistant" by:
1.  **Multi-Dimensional Analysis**: Adding RSI and Volume indicators.
2.  **Visual Confirmation**: Integrating real-time K-line charts.
3.  **Cloud-Synced Mock Trading**: Implementing a portfolio system that works across all user devices using a cloud database.
4.  **Plain-Language Insights**: Translating complex math into actionable human advice.

## 3. Goals
- Increase user confidence in data analysis through multi-indicator confirmation.
- Provide a "low-stakes" environment for practicing trading.
- Ensure a seamless experience between desktop and mobile.
